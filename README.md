## Description

- The utility finds and shows differences in configuration files. 
- Works with .json, .yml, .ini files.
- Three formats of output are available: 'tree' as default, 'plain' and 'json'.

## Install

`$ npm i -g gendiff-edu`

## Usage

`$ gendiff [options] <firstConfig> <secondConfig>`
